## Quick start

1. Upload all files to your `mma12027.github.io` repo
2. Upload your profile photo as `photo.jpg`
3. GitHub Pages will auto-deploy — visit `https://mma12027.github.io`

## Editing

- Search for `EDIT:` comments in the HTML to find placeholders
- Add publications by copying a `pub-entry` block in `Publications.html`
- Add talks/events by copying an `event-row` block in `Events.html`
- Add blog posts by copying a `blog-card` block in `Blog.html`
- Colors and fonts can be changed in `style.css` under `:root`
